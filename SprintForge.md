## Features

| Tab          | What's inside |
|---|---|
| **Overview** | KPI cards · burndown chart · team workload · status breakdown |
| **Backlog**  | Priority-grouped tasks · add task form · assign to sprints |
| **Sprint Plan** | Drag tasks Backlog ↔ Sprint 1 ↔ Sprint 2 |
| **Kanban**   | Drag-and-drop board across 4 columns |
| **Team**     | Members table · workload charts · role editing · invite by email |
| **Docs**     | Create PRD, Sprint Plan, Retrospective documents |
| **Timeline** | Milestones, phases, releases, deadlines with progress bars |
